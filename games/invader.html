画像 / 音 差し替え

ここにファイルをドラッグ＆ドロップすると、その場で見た目や音を変更できます。
永続反映したい場合 は CMS で data/invader-config.json と data/sounds.json を更新してください。

差し替え対象
ここにドロップ
画像なら PNG/JPG/WebP、音なら MP3/WAV/OGG/M4A
CMS常設変更: Pages CMS で JSON の画像・音パスを更新
その場確認: このページ右側のドロップで即差し替え
プレイ操作: 左右キー / A,D / スペース / スマホボタン / 横ドラッグ
まだ差し替えはありません。
